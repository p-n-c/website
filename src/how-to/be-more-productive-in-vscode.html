Be more productive in VSCode

Shortcuts

  • Change both opening and closing HTML tags
    1. F2
    2. and then write the new tag
  • Choose the language of a new file
    1. Ctrl/⌘ + N: Create new file
    2. Ctrl/⌘ + K, M: Open the language mode menu
  • Open the default terminal
    Ctrl/⌘ + ': Open the default terminal
  • Surround a selected piece of text with a new tag
    1. Ctrl/⌘ + Shift + P: Open the command palette
    2. Start typing to find: Emmet: Wrap with Abbreviation
    3. Enter
    4. Type the tag without the angle brackets, e.g. div
    5. Enter
    The last command you called from the palette will be directly accessible with Ctrl/⌘ + Shift + P, Enter
  • Zen Mode - Full screen, no sidebars
    Ctrl + K, Z
  • Navigation between panes
    • Ctrl/⌘ + 0: Left sidebar
    • Ctrl/⌘ + 1: Main editor window
    • Ctrl/⌘ + ': Terminal
