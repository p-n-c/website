Auto assign

This action uses Auto Assign Action

An action which adds reviewers to the pull request when the pull request is opened.

To create a list of accepted reviewers and auto-assign PRs to their author:

  1. Go to ~/actions
  2. Click on New workflow
  3. Click on set up a workflow for yourself
  4. Create 2 yml workflows. See Auto Assign Action for yml specifics.
    1. actions.yml
    2. auto_assign.yml

When a PR is made, the author is assigned and a list of reviewers generated.
