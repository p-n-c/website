How we work

Visitors spend most of their time on other sites. We like to do the same, using browser extensions and npm packages.

Few dependencies, no surprises

This site is simple because we didn't want to spend time designing it. It is written in semantic, valid HTML, and styled using a small amount of CSS. We don't use JavaScript to create pages but we do use it to enhance them.

We keep our developer dependencies to a minimum.

There are no client side dependencies other than for specific use cases.
